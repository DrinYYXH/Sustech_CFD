**布局：** 左公式右参数

**左侧 — 控制方程与初值：**

$$\frac{\partial u}{\partial t} + a\frac{\partial u}{\partial x} = 0, \quad a = 1, \quad x \in [-1, 1]$$

$$u(x,0) = \begin{cases} 1.0, & -0.5 < x < 0.5 \\ 0.0, & \text{otherwise} \end{cases}$$

- 解析解：$u(x,t) = u_0(x-t)$，周期延拓
- t=2.0 和 t=8.0 时波走完 1 个 / 4 个整周期 → 解与初值重合

**右侧 — 参数表：**

| 参数 | 值 |

| CFL | 0.5, 0.8 |
| Δt (CFL=0.8) | 0.01 |
| Δt (CFL=0.5) | 0.00625 |

> 两种 CFL 均满足稳定性条件 ν ≤ 1。

### 第 3 页：数值方法

**布局：** 上方大公式，下方三列表格

**核心公式 — 单元线性重构推进格式（项目 Eq. 3）：**

$$\boxed{u_j^{n+1} = u_j^n - \nu\left(u_j^n - u_{j-1}^n\right) - \frac{\nu(1-\nu)}{2}\left(\delta_j - \delta_{j-1}\right)}, \quad \nu = \frac{a\Delta t}{\Delta x}$$

其中 $\delta_j = \sigma_j \Delta x$ 为单元 $j$ 内的受限斜率。

**公式结构拆解：**
- $-\nu(u_j^n - u_{j-1}^n)$：一阶迎风通量（保证基本输送）
- $-\frac{\nu(1-\nu)}{2}(\delta_j - \delta_{j-1})$：反扩散修正项（提升至二阶精度）
- $\delta_j$：斜率项——**三种格式的唯一区别**

**三种格式（三列并排）：**

| | Lax-Wendroff | van Leer | SUPERBEE |
|---|:---:|:---:|:---:|
| $\delta_j=$ | $u_{j+1}-u_j$ | $\phi_{\text{VL}}(r_j)(u_{j+1}-u_j)$ | $\phi_{\text{SB}}(r_j)(u_{j+1}-u_j)$ |
| $\phi(r)=$ | 1（无限制） | $\dfrac{r+\vert r\vert}{1+\vert r\vert}$ | $\max(0,\min(2r,1),\min(r,2))$ |
| 性质 | 线性、二阶 | 非线性、TVD | 非线性、TVD（最陡） |

其中 $r_j = \dfrac{u_j - u_{j-1}}{u_{j+1} - u_j}$（局部斜率比）

**配图建议：** 右下角放 Sweby 图，标注 VL 和 SB 的位置。

**讲稿要点：**
- 三种格式共用同一个推进公式 (Eq. 3)，代码只需切换 $\delta_j$ 的计算——体现了格式设计的系统性
- 限制器的核心作用：在间断附近用非线性方式压缩斜率（保单调），在光滑区恢复二阶精度
- LW 是无限制器的特例（$\phi \equiv 1$），这正是它产生振荡的根源

### 第 6 页：TV 分析与格式评价

**布局：** 左侧 TV 图，右侧表格 + 分析

**上半部分 — TV 定义与意义：**

$$\text{TV}(u) = \sum_i |u_{i+1} - u_i| \quad\text{（周期域含卷绕项 } |u_1 - u_N|\text{）}$$

> TV 衡量解的总振荡量。单调初值 → 解析解的 TV 守恒。数值解的 TV 若增长 = 格式产生了新的极值。

**左侧：** 放 `total_variation.png`（占 50% 宽度）

**右侧表格：**

| 格式 | TV(t=0) | TV(t=2) | TV(t=8) | 单调？| TVD？|
|------|:---:|:---:|:---:|:---:|:---:|
| LW | 2.00 | 3.14 | 3.56 | ✗ | ✗ |
| van Leer | 2.00 | 2.00 | 2.00 | ✓ | ✓ |
| SUPERBEE | 2.00 | 2.00 | 2.00 | ✓ | ✓ |

**底部一行关键结论：**
> **TVD 格式 = 自动保单调 = 不会产生新的局部极值 = 无振荡**

**讲稿要点：**
- LW 的 TV 单调递增：每过一个间断，振荡幅度增大一次——格式在设计上就没有抑制振荡的机制
- VL 和 SB 的 TV 严格持平：限制器在每步把解约束在 TVD 范围内
- TVD 是比"看起来没振荡"更强的保证——它是数学上可证的单调性

### 第 7 页：物理解释 — 为何 LW 振荡？

**这是展示理解深度的核心页，建议给 2 分钟。**

**布局：** 三层递进结构

**第一层 — Taylor 展开 → 修正方程（上半部分左侧）：**

将 LW 格式 Taylor 展开，利用 PDE 消去时间导数，得修正方程：

$$\frac{\partial u}{\partial t} + a\frac{\partial u}{\partial x} = \underbrace{-\frac{a\Delta x^2}{6}(1-\nu^2)}_{\text{色散系数}} \frac{\partial^3 u}{\partial x^3} + \mathcal{O}(\Delta x^3)$$

- 主导误差：**三阶色散项** $\partial^3 u/\partial x^3$（奇数阶）
- **没有** $\partial^2 u/\partial x^2$ 项——LW 巧妙抵消了二阶耗散项才达到二阶精度
- 色散 → 不同波数分量以不同速度传播 → 高频分量在间断前后堆积 → **$2\Delta x$ 波长振荡**

**第二层 — 对比一阶迎风（上半部分右侧）：**

$$\frac{\partial u}{\partial t} + a\frac{\partial u}{\partial x} = \underbrace{\frac{a\Delta x}{2}(1-\nu)}_{\text{正耗散系数}} \frac{\partial^2 u}{\partial x^2} + \mathcal{O}(\Delta x^2)$$

- 主导误差：**二阶耗散项** $\partial^2 u/\partial x^2$（偶数阶）
- 正耗散 → 抹平间断，但无振荡
- 代价：一阶精度 $\mathcal{O}(\Delta x)$

**第三层 — Godunov 定理的归结（底部方框）：**

> **Godunov 定理 (1959)：单调的线性格式至多为一阶精度。**
>
> LW 是**线性** + **二阶** → 必定**不单调**（振荡）。
> van Leer / SUPERBEE 是**非线性**（$\phi$ 依赖 $r_j$ → 依赖解本身）→ 可以**二阶** + **单调**。

**衔接过渡箭头：**
```
一阶迎风（正耗散，一阶，单调）
       ↓  加上反扩散修正
LW（零耗散，二阶，不单调 → 振荡）
       ↓  加上非线性限制器
van Leer / SUPERBEE（局部耗散，二阶光滑区，单调）
```

**讲稿要点：**
- 这是整个 Part I 的理论核心——用 Taylor 展开解释 LW 的振荡，用 Godunov 定理解释为什么需要限制器
- 修正方程的推导展示在作业答案文档中，PPT 上只放结论
- 关键洞察：偶阶导数 = 耗散（抹平），奇阶导数 = 色散（振荡）。LW 只有奇阶误差。

### 第 8 页：小结

**布局：** 表格 + 一句话结论

**核心对比表：**

| | Lax-Wendroff | van Leer | SUPERBEE |
|---|:---:|:---:|:---:|
| 精度 | 二阶（全局） | 二阶（光滑区）/ 一阶（极值点） | 二阶（光滑区）/ 一阶（极值点） |
| 单调性 | ✗（Gibbs 振荡） | ✓ | ✓ |
| TVD | ✗ | ✓ | ✓ |
| 间断分辨率 | 有振荡 | ~7 格 | **~5 格（最陡）** |
| 限制器光滑性 | — | 光滑（$C^1$） | 分段（$r=1/2, 2$ 处有角点） |
| 主导误差 | 色散（$\partial^3 u/\partial x^3$） | 局部耗散（$r \neq 1$ 时） | 局部耗散 + 反扩散（$r>1$ 时） |
| 适用场景 | 光滑解 | 通用稳健 | 需要最锐利间断 |

**底部结论（加粗框）：**

> **纯线性二阶格式无法同时实现高精度与无振荡。通过非线性限制器在光滑区保留二阶精度、在间断附近局部降阶耗散，是突破 Godunov 定理约束的关键。**

**讲稿要点：**
- 三种格式在同一框架下仅 δ 不同，但行为差异巨大——体现了数值方法的精巧之处
- 没有"最好"的格式：LW 对光滑解最优，van Leer 最稳健，SUPERBEE 间断最锐利
