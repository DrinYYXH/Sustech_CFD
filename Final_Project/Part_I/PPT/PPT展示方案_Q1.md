### 第 3 页：数值方法

**布局：** 左侧公式推导，右侧三列表格；下半部分格式等价说明

**上半部分 — 实现格式（单元线性重构形式，项目 Eq. 3）：**

$$\boxed{u_j^{n+1} = u_j^n - \nu\left(u_j^n - u_{j-1}^n\right) - \frac{\nu(1-\nu)}{2}\left(\delta_j - \delta_{j-1}\right)}, \quad \nu = \frac{a\Delta t}{\Delta x}$$

其中 $\delta_j$ 为单元 $j$ 内的受限斜率。

**公式结构拆解：**
- $-\nu(u_j^n - u_{j-1}^n)$：一阶迎风通量
- $-\frac{\nu(1-\nu)}{2}(\delta_j - \delta_{j-1})$：反扩散修正项
- $\delta_j$：斜率——**三种格式的唯一区别**

**三种格式（三列并排）：**

| | Lax-Wendroff | van Leer | SUPERBEE |
|---|:---:|:---:|:---:|
| $\delta_j=$ | $u_{j+1}-u_j$ | $\phi_{\text{VL}}(r_j)(u_{j+1}-u_j)$ | $\phi_{\text{SB}}(r_j)(u_{j+1}-u_j)$ |
| $\phi(r)=$ | 1（无限制） | $\dfrac{r+\vert r\vert}{1+\vert r\vert}$ | $\max(0,\min(2r,1),\min(r,2))$ |
| 性质 | 线性、二阶 | 非线性、TVD | 非线性、TVD（最陡） |

其中 $r_j = \dfrac{u_j - u_{j-1}}{u_{j+1} - u_j}$

**下半部分 — 与课堂格式的等价关系：**

> 课堂教授格式为**守恒型通量形式**：$u_j^{n+1} = u_j^n - \frac{\Delta t}{\Delta x}(F_{j+1/2} - F_{j-1/2})$，其中 $F_{j+1/2} = a u_j + \frac{a}{2}(1-\nu)\delta_j$。将通量代入即可化为本页上半部分的单元重构形式。两者**数学完全等价**。

**配图：** 见 `ppt_format_equivalence.png`（由 `plot_ppt_figures.py` 生成），展示从通量形式到单元重构形式的推导流程。可在讲解时切换到此图。

**讲稿要点：**
- 课堂教的通量守恒形式和项目使用的单元重构形式是同一公式的两种写法——代入即得
- 本项目选择单元重构形式编码是因为它更紧凑（单行、无中间通量数组），但两种形式完全可以互换
- 三种格式共用同一条推进公式，代码只需切换 $\delta_j$ 的计算
- 限制器的核心作用：在间断附近用非线性方式压缩斜率（保单调），在光滑区恢复二阶精度
- LW 是无限制器的特例（$\phi \equiv 1$），这正是它产生振荡的根源

### 第 7 页：物理解释 — LW 的两个缺陷：振荡 vs 不单调

**这是展示理解深度的核心页，建议给 2 分钟。**

**布局：** 左右双栏，分别解释两个独立机制

---

**左侧栏 — 机制一：振荡（Gibbs 现象）**

**根源：修正方程的色散主导**

Taylor 展开 LW 格式，利用 PDE 消去时间导数，得修正方程：

$$\frac{\partial u}{\partial t} + a\frac{\partial u}{\partial x} = \underbrace{-\frac{a\Delta x^2}{6}(1-\nu^2)}_{\text{色散系数}} \frac{\partial^3 u}{\partial x^3} + \mathcal{O}(\Delta x^3)$$

- 主导误差是**三阶色散项** $\partial^3 u/\partial x^3$（奇数阶）
- 不同波数分量以不同相速度传播 → 高频分量在间断前后堆积
- 表现为间断附近的 **$2\Delta x$ 波长锯齿状振荡**
- 对比：一阶迎风的修正方程是 $\partial^2 u/\partial x^2$ 耗散主导（抹平、无振荡）

> **色散 → 振荡的形状（wiggles），但不直接导致新极值。**

---

**右侧栏 — 机制二：不单调（新极值产生）**

**根源：线性二阶格式积分模板中的负系数**

LW 的一次推进等效于对三个模板点 $u_{j-1}, u_j, u_{j+1}$ 做加权组合：

$$u_j^{n+1} = \underbrace{\frac{\nu(1+\nu)}{2}}_{\text{上游权重 }>0}u_{j-1}^n + \underbrace{(1-\nu^2)}_{\geq 0}u_j^n \underbrace{- \frac{\nu(1-\nu)}{2}}_{\text{下游权重 }<0}u_{j+1}^n$$


**三个权重的和为 1：** $\frac{\nu(1+\nu)}{2} + (1-\nu^2) - \frac{\nu(1-\nu)}{2} \equiv 1$（任意守恒格式的必要条件，直接验证即得）。**但这不构成"加权平均"——加权平均要求所有权重 ≥0，而下游点 $u_{j+1}$ 的权重恒为负。**

**什么是负系数？**

当 $\nu \in (0,1)$ 时，下游点 $u_{j+1}$ 的权重恒为 $-\frac{\nu(1-\nu)}{2} < 0$。物理含义：反扩散修正项 $\frac{\nu(1-\nu)}{2}(\delta_j - \delta_{j-1})$ 试图抵消一阶迎风的耗散，但**过度补偿**了——它不仅抵消了数值耗散，还对下游方向施加了反号贡献，相当于用"负的权重"从 $u_{j+1}$ 中减去一部分值。

**数值示例（$\nu = 0.8$，权重：$0.72, 0.36, -0.08$）：**

| $[u_{j-1}, u_j, u_{j+1}]$ | $u_j^{n+1}$ = | 结果 | 问题 |
|------|------|:---:|------|
| $[0, 0, 1]$ | $0.72\cdot 0 + 0.36\cdot 0 - 0.08\cdot 1$ | **−0.08** | 负下冲！输入全 ≥0，输出 <0 |
| $[1, 1, 0]$ | $0.72\cdot 1 + 0.36\cdot 1 - 0.08\cdot 0$ | **1.08** | 正超调！输入全 ≤1，输出 >1 |

两个例子中输入均在 $[0,1]$，输出却超出此区间——**一次推进即产生新极值**。

**为何负系数必然导致新极值？——凸组合 vs 非凸组合**

- 权重和为 1 = 格式的**守恒性**（consistency），所有守恒格式都满足
- 所有权重 ≥0 = 格式的**单调性**（monotonicity），只有一阶格式才满足（Godunov 定理）
- **凸组合** = 权重和 1 + 所有权重 ≥0 → 输出 $\in [\min(\text{输入}), \max(\text{输入})]$ → 保单调
- LW：权重和 1 ✓，但下游权重 <0 ✗ → **非凸组合** → 输出可以低于所有输入或高于所有输入

**对比一阶迎风（$\delta_j = 0$）：**

$$u_j^{n+1} = \nu \cdot u_{j-1}^n + (1-\nu) \cdot u_j^n$$

两个系数均 ≥0 且和为 1 → **严格凸组合** → 永远保单调。代价：截断误差 $\mathcal{O}(\Delta x)$。

> **Godunov 定理 (1959)：单调（保极值）的线性格式至多为一阶精度。**
>
> LW 是**线性** + **二阶** → 系数中必然出现负数（否则不可能抵消 $\mathcal{O}(\Delta x)$ 耗散） → **必定不单调**（产生新极值，并逐层扩散为振荡）。

---

**推广：带限制器的一般情况（$\phi(r) = c$）**

将 $\delta_j = c \cdot (u_{j+1} - u_j)$ 代入统一推进公式，得到**以 $c$ 为参数的广义三系数模板**：

$$\boxed{u_j^{n+1} = \underbrace{\frac{\nu(2 - c + c\nu)}{2}}_{w_{-1}(c)} u_{j-1}^n \;+\; \underbrace{\big[1 - \nu + c\nu - c\nu^2\big]}_{w_0(c)} u_j^n \;-\; \underbrace{\frac{c\nu(1-\nu)}{2}}_{w_{+1}(c)} u_{j+1}^n}$$

**验证：** $w_{-1} + w_0 + w_{+1} \equiv 1$（对任意 $c$ 成立——守恒性不变）。

**配图：** 见 `ppt_weight_signs.png`（四联子图，$c \in [-0.5, 3.5]$，RdBu 渐变色）。前三个子图分别显示 $w_{-1}$、$w_0$、$w_{+1}$ 的连续值（蓝 = 负值，红 = 正值，白 = 零，黑色实线为零等值线）。第四子图显示 $\min(w_{-1}, w_0, w_{+1})$——即该 $(c,\nu)$ 处"最负的权重"。蓝区 = 存在负权重（非凸组合），白/红区 = 三权重全正（凸组合）。清晰可见：$\nu < 1$ 时凸组合仅见于 $c \leq 0$ 的窄带。

**各 $c$ 值下的三系数（$\nu = 0.8$）：**

| $c$ | $w_{-1}$（上游） | $w_0$（中心） | $w_{+1}$（下游） | 和 | 凸组合？ | 对应格式 |
|:---:|:---:|:---:|:---:|:---:|:---:|------|
| 0 | 0.80 | 0.20 | 0.00 | 1 | ✓ | 一阶迎风 |
| 0.5 | 0.76 | 0.28 | −0.04 | 1 | ✗ | 弱反扩散 |
| 1.0 | 0.72 | 0.36 | −0.08 | 1 | ✗ | LW |
| 1.5 | 0.68 | 0.44 | −0.12 | 1 | ✗ | 强反扩散 |
| 2.0 | 0.64 | 0.52 | −0.16 | 1 | ✗ | SB 上限 |

**核心——$w_{+1}(c)$ 是理解限制器的代数钥匙：**

$$w_{+1}(c) = -\frac{c\nu(1-\nu)}{2}$$

- 唯一可能为负的系数（$w_{-1} \geq 0$ 和 $w_0 \geq 0$ 在 $\nu \leq 1$ 时始终成立）
- **$c$ 越大，下游负权重的绝对值越大** → 反扩散越激进 → 突破 $[u_{\min}, u_{\max}]$ 的风险越高
- $c = 0$ 时 $w_{+1} = 0$ → 退还为一阶迎风，恢复凸组合，严格保单调
- **限制器的本质：在每个网格局部动态选择一个 $c$，使局部 $w_{+1}$ 的负效应被 TVD 条件约束在安全范围内**

这直接解释了限制器的一切行为：
- VL/SB 在间断附近将 $c \to 0$ → 关闭 $w_{+1}$ 负值 → 恢复局部单调
- SUPERBEE 在 Sweby 区域内取最大的 $c$（接近 2）→ $|w_{+1}|$ 最大化 → 最强反扩散 → 间断最陡
- TVD 约束的代数含义：保证由 $w_{+1} < 0$ 注入的新极值不会被格式逐层放大

**追问：既然 $c \leq 0$ 都能使三权重非负，随便取一个 $c < 0$ 不就能消除振荡了吗？**

不行。问题从单调性转移到了**精度**——回顾分段修正方程的耗散系数：

$$D(c) = \frac{a\Delta x}{2}(1-\nu)(1-c)$$

| $c$ | $D(c)$ | 耗散强度 | 精度 |
|:---:|------|:---:|:---:|
| $c < 0$ | $> \frac{a\Delta x}{2}(1-\nu)$ | **超量耗散** | 低于一阶 |
| $c = 0$ | $\frac{a\Delta x}{2}(1-\nu)$ | 一阶迎风耗散 | $\mathcal{O}(\Delta x)$ |
| $c = 1$ | 0 | 零耗散 | $\mathcal{O}(\Delta x^2)$ |

- $c < 0$：三权重确实全正 → 单调。但 $D(c)$ 比一阶迎风还大 → **过度耗散，精度比一阶还差**——解被过度抹平
- $c = 0$：权重全正 + 耗散最小 → 这是线性单调格式能达到的**最优精度**（Godunov 定理的另一种表述：线性单调格式的最高精度是一阶）
- $c > 0$（如 LW 的 $c=1$）：$D(c)$ 减小 → 精度提升，但 $w_{+1}$ 变负 → 不单调

**本质：单调性和精度在 $c$ 轴上相互排斥。**

```
c < 0          c = 0          0 < c < 1        c = 1          c > 1
  ← 过度耗散     最优单调       耗散不足       零耗散二阶      反扩散
  ← 全正单调     全正单调       w_{+1}<0       w_{+1}<0       w_{+1}更负
  ← 精度<一阶    精度=一阶      精度~一阶       精度=二阶      精度~一阶
```

**限制器的意义：** 不在全局选一个固定的 $c$，而是在每个网格局部自适应地选——光滑区选 $c \approx 1$（二阶），间断附近选 $c \approx 0$（单调）。**非线性是唯一能同时获取两个区域最优值的途径。**

---

**从权重分析到限制器设计——自然的推导链：**

回顾下游权重 $w_{+1}(c) = -\frac{c\nu(1-\nu)}{2}$。当 $u_{j+1}$ 很大（间断来临）时，负权重乘以大值 → $u_j^{n+1}$ 被强力下拉 → 产生下冲。**调制策略：让 $c$ 随局部流态自适应——**

$$c_j = \phi(r_j), \quad r_j = \frac{u_j - u_{j-1}}{u_{j+1} - u_j}$$

**$r_j$ 的物理直觉：**

| $r_j$ | 流态 | 需要的 $c$ | 原因 |
|:---:|------|:---:|------|
| $r_j < 0$ | 局部极值（锯齿振荡） | $c = 0$ | $u_{j+1}$ 与 $u_{j-1}$ 异号——$w_{+1}$ 的负效应最危险，必须完全关闭 |
| $r_j \approx 1$ | 光滑单调（三网格共线） | $c = 1$ | $u_{j+1}$ 与邻居协调——$w_{+1}$ 的负值不会制造意外，可放心用 LW 二阶 |
| $r_j \to \infty$ | 陡峭梯度（$u_{j+1} \approx u_j$） | $c \to 2$ | $u_{j+1}-u_j \approx 0$ → $w_{+1} \approx 0$ 自动安全，可以加大 $c$ 锐化梯度 |

**配图：** 见 `ppt_r_sensor.png`——三行图展示方波初值下 $u(x)$、$r_j$（对数坐标）、及自适应 $c_j = 1-\theta_j$ 沿 $x$ 的分布。$r_j$ 在间断处 $\to \pm\infty$，在常数区无定义 (0/0)；而 $c_j$ 用曲率传感器自动在间断处降至 ≈0、常数区恢复 ≈1，无需处理比值病态问题。

**这个 $c_j = \phi(r_j)$ 正是限制器函数。** 由此 $\phi(r)$ 必须满足的约束自然浮现：

$$\boxed{\phi(r) = 0 \;\; (r \leq 0), \qquad \phi(1) = 1, \qquad 0 \leq \phi(r) \leq 2, \qquad 0 \leq \frac{\phi(r)}{r} \leq 2}$$

第一项：极值点关闭反扩散（$c=0$）。第二项：光滑区恢复 LW（$c=1$）。后两项：TVD 约束（Sweby 区域）。van Leer 和 SUPERBEE 是满足此约束的两个具体实例——它们的不同仅在于光滑区到极值点的过渡方式（平缓 vs 激进）。

> **从 $w_{+1}(c)$ 的负权重问题 → $c$ 的节制需求 → 局部斜率比 $r_j$ 作为传感器 → $\phi(r)$ 的设计约束 → VL/SB 的具体公式。限制器不再是一个任意的"黑箱修正"，而是从代数和物理双重考量中自然涌现的必然构造。**

---

**另一条路：不用 $r_j$ 传感器，直接用解的曲率自适应调制 $c$**

VL/SB 的传感器 $r_j = (u_j-u_{j-1})/(u_{j+1}-u_j)$ 有两个固有问题：(1) 分母接近零时数值病态；(2) $r_j$ 是比值，物理含义不够直接。

**替代方案——基于二阶差商的传感器：**

定义一个直接度量解的局部"光滑度"的指标，不依赖比值：

$$\theta_j = \frac{|u_{j+1} - 2u_j + u_{j-1}|}{|u_{j+1} - u_j| + |u_j - u_{j-1}| + \varepsilon_0}$$

其中 $\varepsilon_0$ 是一个小量（如 $10^{-12}$），仅用于防止分母为零。

- **分子** $|u_{j+1} - 2u_j + u_{j-1}|$：二阶差商 → 度量解的局部曲率 → 间断处大、光滑处趋零
- **分母**：一阶差商之和 → 归一化，使 $\theta_j \in [0, 1]$
- **$\varepsilon_0$**：纯粹的数值保护，不影响 $\theta_j$ 的物理含义

**自适应的 $c_j$：**

$$c_j = 1 - \theta_j$$

| 区域 | $\theta_j$ | $c_j$ | $w_{+1}(c_j) = -\frac{c_j\nu(1-\nu)}{2}$ | 效果 |
|------|:---:|:---:|------|------|
| 光滑区 | $\to 0$ | $\to 1$ | $-\frac{\nu(1-\nu)}{2}$（LW 负权重） | 二阶精度，$u_{j+1}$ 与邻点协调，负权重不制造意外 |
| 间断附近 | $\to 1$ | $\to 0$ | $0$ | 自动退还一阶迎风，$w_{+1}$ 归零，恢复凸组合 |
| 过渡区 | $\in (0,1)$ | $\in (0,1)$ | 介于两者之间 | 平滑过渡，无硬切换 |

**与 VL/SB 限制器的本质区别：**

| | VL / SB 限制器 | 曲率传感器（ALW 型） |
|---|:---:|:---:|
| **传感器** | $r_j = \frac{u_j-u_{j-1}}{u_{j+1}-u_j}$（比值） | $\theta_j$（二阶差商，绝对值） |
| **数值稳定性** | $u_{j+1} \approx u_j$ 时 $r_j$ 无界 | $\theta_j$ 始终 $\in [0,1]$，无条件有界 |
| **设计逻辑** | 通过 Sweby 区域约束保证 TVD | 直接度量曲率 → 曲率大 = 需要耗散 |
| **$c=1$ 的恢复** | $r_j=1$ 时 $\phi=1$（三网格共线） | $\theta_j=0$ 时 $c=1$（曲率为零） |
| **参数** | 无自由参数 | 仅一个保护小量 $\varepsilon_0$，物理上可忽略 |

**这个方案的优势：** 不需要构造 $\phi(r)$ 函数族、不需要 Sweby 图、不需要 TVD 证明——从 $w_{+1}(c) < 0$ 的问题出发，用解的局部曲率直接判断"这里负权重会不会惹祸"，自适应地衰减 $c$。这就是本项目 Task 5 中 **Adaptive LW** 的设计思想。

---

**两个机制的区分与关联：**

| | 振荡（Oscillation） | 不单调（Non-monotonicity） |
|---|---|---|
| **根源** | 修正方程的色散项 $\partial^3 u/\partial x^3$ | 积分模板中的负系数 |
| **表现** | 间断附近的 $2\Delta x$ 锯齿 | 解超出 $[u_{\min}, u_{\max}]$ 的新极值 |
| **分析方法** | Taylor 展开 → 修正方程 | Godunov 定理 / Harten TVD 理论 |
| **VL/SB 如何解决** | 在间断附近降为一阶迎风（局部耗散） | 非线性限制器 → TVD → 自动保单调 |

**衔接过渡：**
```
一阶迎风（耗散 ∂²，一阶，单调——系数全正）
       ↓  加反扩散修正（抵消耗散，提升精度）
LW（色散 ∂³，二阶，不单调——出现负系数）
       ↓  加非线性限制器（局部控制负系数效应）
van Leer / SUPERBEE（自适应耗散，二阶光滑区，TVD 保单调）
```

**讲稿要点：**
- 强调两个缺陷的独立性：即使不存在间断，LW 在光滑极值点也会产生新极值（不单调）；而色散振荡主要在间断附近明显
- 偶阶导数 = 耗散（抹平），奇阶导数 = 色散（振荡）。LW 只有奇阶误差 → 振荡但不耗散
- Godunov 定理的核心：线性 + 二阶 → 系数必出现负数 → 不单调。这是结构性的，无法通过调参数规避

|---|:---:|:---:|:---:|

### 第 9 页：Task 3 — 精度阶数研究 · 理论方法

**布局：** 上半部分 LW 修正方程推导概要，下半部分 VL/SB 分段分析框架

---

**上半部分 — LW 修正方程推导（修正方程法三步）：**

**Step 1 — Taylor 展开：** 将 $u_j^{n+1}, u_{j\pm1}^n$ 在 $(x_j, t^n)$ 处展开至 $\mathcal{O}(\Delta x^3, \Delta t^3)$。

**Step 2 — 消去时间导数：** 利用原 PDE $u_t = -au_x$ 导出 $u_{tt} = a^2 u_{xx},\; u_{ttt} = -a^3 u_{xxx}$，将所有时间导数替换为空间导数。

**Step 3 — 整理修正方程：**

$$\boxed{\frac{\partial u}{\partial t} + a\frac{\partial u}{\partial x} = -\frac{a\Delta x^2}{6}(1-\nu^2)\frac{\partial^3 u}{\partial x^3} + \mathcal{O}(\Delta x^3)}$$

→ 截断误差 $\tau_{\text{LW}} \sim \mathcal{O}(\Delta x^2)$：**LW 为二阶精度。** 主导误差为三阶色散项（奇数阶）——没有 $\partial^2 u/\partial x^2$ 耗散项。

**对比一阶迎风（极值点退化情形）：**

$$\boxed{\frac{\partial u}{\partial t} + a\frac{\partial u}{\partial x} = \frac{a\Delta x}{2}(1-\nu)\frac{\partial^2 u}{\partial x^2} + \mathcal{O}(\Delta x^2)}$$

→ 截断误差 $\sim \mathcal{O}(\Delta x)$：**一阶精度。** 主导误差为二阶耗散项（偶数阶，正系数 → 抹平、无振荡）。

---

**下半部分 — VL/SB 的分段修正方程分析（关键理论框架）：**

限制器格式非线性 → 不能全局 Taylor 展开。将 $\phi(r)=c$ 冻结为常数，$\delta_j = c(u_{j+1}-u_j)$ 退化为**以 $c$ 为参数的广义修正方程**：

$$\boxed{\frac{\partial u}{\partial t} + a\frac{\partial u}{\partial x} = \underbrace{\frac{a\Delta x}{2}(1-\nu)(1-c)}_{\text{耗散系数 } D(c)} \frac{\partial^2 u}{\partial x^2} \;-\; \frac{a\Delta x^2}{6}(1-\nu^2)\frac{\partial^3 u}{\partial x^3} + \mathcal{O}(\Delta x^3)}$$

**核心观察：**
- 耗散系数 $D(c) = \frac{a\Delta x}{2}(1-\nu)(1-c)$ — **依赖于限制器输出 $c$**
- 色散项 $-\frac{a\Delta x^2}{6}(1-\nu^2)u_{xxx}$ — **与 $c$ 无关，始终存在**

| 流态 | $c$ | $D(c)$ | 局部精度 | 物理行为 |
|------|:---:|------|:---:|------|
| 极值点 ($r \leq 0$) | 0 | $\frac{a\Delta x}{2}(1-\nu) > 0$ | **1 阶** | 正耗散（扩散），压制振荡 |
| 光滑单调区 ($r \approx 1$) | 1 | 0 | **2 阶** | 纯色散，等价于 LW |
| SB 压缩区 ($r > 1$) | 1–2 | **负值** | 1 阶 | **反扩散**（锐化梯度），使间断更陡 |

> **关键洞察：** $c=1$ 是光滑解的**吸引子**——网格加密时 $r \to 1$，绝大多数网格运行在 $c \approx 1$（二阶）。$L_\infty$ 降阶仅来自极少数极值点的 $c \to 0$ 退化。

**配图：** `convergence_l1.png` 和 `convergence_linf.png`

**数值验证 — 网格加密法：**

光滑初值 $\sin(\pi x)$，CFL=0.8，t=1.0，N = 20, 40, 80, 160, 320。

| 格式 | L₁ 收敛阶 | L∞ 收敛阶 | 理论预测 |

**为什么 L₁ 二阶而 L∞ 降阶？——两组范数的不同视角：**

| | L₁ 范数（全局平均） | L∞ 范数（局部最差） |
|---|------|------|
| 受什么影响 | 所有网格的误差平均值 | 单个误差最大网格 |
| VL/SB 极值点降阶的影响 | 极小——极值点仅占万分之几的网格 | 极大——一个降阶点即主导整个范数 |

- VL 限制器 $C^1$ 光滑 → 极值点附近的 $\phi \to 0$ 过渡更平缓 → L∞ ≈ 1.39
- SB 限制器分段 → $r=1/2$ 和 $r=2$ 处有角点 → 极值点截断更刚硬 → L∞ ≈ 1.10

---

**归结为 Godunov 定理的完整图景：**

| | LW | VL / SB |
|---|:---:|:---:|
| 格式类型 | 线性 | 非线性（$\phi$ 依赖 $r_j$ → 依赖解） |
| 精度 | 全局二阶 | 光滑区二阶 / 极值点一阶 |
| 单调性 | ✗ | ✓（TVD） |
| Godunov 定理 | 线性 + 二阶 → 不单调 | 非线性 + TVD → 可二阶 + 单调 |

> **核心结论：** 极值点降阶不是 bug，而是 TVD 格式为突破 Godunov 定理**必须付出的代价**。限制器的非线性使 $c$ 随流态自适应——需要耗散处（$c \to 0$）压制振荡，光滑处（$c \to 1$）恢复精度。绝大多数网格 $c \to 1$（L₁ 二阶），少数极值点 $c \to 0$（L∞ 降阶）。

**讲稿要点：**
- L₁ 和 L∞ 的分化是 TVD 格式"花钱买单调"的直接证据
- 分段修正方程的统一框架优雅地解释了表中所有数字
- Godunov 定理不仅说"线性二阶不可能单调"，还精确指出了解决路径：非线性

### 第 10b 页：Task 3 补充 — 方波初值下的收敛率退化

**说明：** 本页为对比/补充页——若时间有限可跳过，但强烈建议保留以展示对精度分析方法的深入理解。

**布局：** 上半部分展示方波收敛结果（斜率 ≈ 1），下半部分对比 sin(πx) 结果（斜率 ≈ 2），中间解释原因。

---

**现象：方波初值下 L₁ / L₂ 收敛率 ≈ 1**

| 初值 | t | L₁ 斜率 | L₂ 斜率 |
|------|:---:|:---:|:---:|
| 方波 | 2.0 | ≈ 1.0 | ≈ 1.0 |
| 方波 | 8.0 | ≈ 1.0 | ≈ 1.0 |

三种格式（LW, VL, SB）在方波初值下收敛阶全部退化至 ≈ 1，与理论二阶相差甚远。

---

**原因分析：解的奇性主导了误差**

数值解的全局误差来自两个部分：

$$\text{Error} = \underbrace{\text{间断附近的误差}}_{\text{由解的奇性决定}} + \underbrace{\text{光滑区的误差}}_{\text{由格式精度决定}}$$

**(1) 间断附近的误差 —— 主导项：**

- 数值格式将零厚度的物理间断抹平为有限过渡区（~5–7 个网格）
- 在过渡区内，每个网格的误差 $\sim \mathcal{O}(1)$
- 过渡区网格数由格式的耗散特性决定，与 $\Delta x$ 无关（始终 ~5–7 格）
- → 过渡区误差贡献 $\sim (\text{常数}) \cdot \Delta x \propto \mathcal{O}(\Delta x)$

**(2) 光滑区的误差：**
- 光滑区内格式为二阶 → 每个网格误差 $\sim \mathcal{O}(\Delta x^2)$
- 但光滑区网格占总数的比例 ≈ 1
- → 光滑区误差贡献 $\sim \mathcal{O}(\Delta x^2)$

**全局 L₁ 误差的主导项：** $\Delta x^1 \gg \Delta x^2$（当 $\Delta x$ 较小时），因此 $\boxed{\text{收敛率} \approx 1}$。

---

**对比：为什么 sin(πx) 初值能恢复二阶？**

| | 方波初值 | sin(πx) 初值 |
|---|:---:|:---:|
| 解的奇性 | 含 $C^{-1}$ 间断 | $C^\infty$ 光滑 |
| 误差主导来源 | 间断附近的 $\mathcal{O}(\Delta x^1)$ | 格式截断误差 $\mathcal{O}(\Delta x^2)$ |
| 各格式的区分度 | 无法区分——都 ≈ 1 | 可区分——LW ≈ 2.0, VL/SB L∞ ≈ 1.1–1.4 |

> **核心结论：** 方波初值下收敛率退化至 ≈ 1 **不是格式精度不够**，而是**解的奇性限制**了所有格式的收敛率。这正是 Task 3 必须使用 sin(πx) 的根本原因——只有当解比格式的截断误差更光滑时，实测收敛阶才能反映格式的真实精度。

**讲稿要点：**
- 方波结果（斜率 ≈ 1）与 sin(πx) 结果（斜率 ≈ 2）的对比，有力地证明了"光滑初值是精度分析的前提"这一方法论原则
- 这一现象对所有格式普适——无论 LW、VL 还是 SB，都受解的奇性约束

### 第 11 页：Task 4 — 数值稳定性 · 理论

**布局：** 上半部分 LW von Neumann 分析，下半部分 VL/SB 的 TVD + 分段 von Neumann 分析

---

**上半部分 — LW 的 von Neumann 分析：**

设 Fourier 模态解 $u_j^n = e^{\sigma n \Delta t} e^{i k j \Delta x}$，代入 LW 格式。一步放大：

$$u_j^{n+1} = e^{\sigma \Delta t} u_j^n$$

代入 LW 两步形式，得一步放大因子：

$$|e^{\sigma\Delta t}|^2 = 1 - 4\nu^2(1-\nu^2)\sin^4\left(\frac{k\Delta x}{2}\right)$$

- 稳定要求 $|e^{\sigma\Delta t}| \leq 1, \forall k$ → 右边第二项必须非负 → $\nu^2(1-\nu^2) \geq 0$ → **$\nu \leq 1$**
- $\nu = 1$：$|e^{\sigma\Delta t}| = 1$，零耗散、零色散，精确传播
- $\nu < 1$：$|e^{\sigma\Delta t}| < 1$（对 $k \neq 0$），格式含数值耗散，但仍有色散

---

**下半部分 — VL/SB 的稳定性：TVD 理论 + 分段 von Neumann 分析**

**(1) TVD 理论（Harten 定理）：**

将格式改写为 Harten 形式：

$$u_j^{n+1} = u_j^n - C_{j-1/2}(u_j^n - u_{j-1}^n) + D_{j+1/2}(u_{j+1}^n - u_j^n)$$

当限制器满足 Sweby 约束（$0 \leq \phi(r) \leq 2$，$0 \leq \phi(r)/r \leq 2$）且 $\nu \leq 1$ 时，系数 $C_{j-1/2} \geq 0$、$D_{j+1/2} \geq 0$、$C_{j-1/2} + D_{j+1/2} \leq 1$ → **格式 TVD** → $L_\infty$ 稳定。

**(2) 分段 von Neumann 分析（关键理论框架）：**

将限制器冻结论为 $\phi(r) = c$（$0 \leq c \leq 2$），$\delta_j = c(u_{j+1}-u_j)$，格式退化为线性。设 $u_j^n = e^{\sigma n\Delta t} e^{ikj\Delta x}$，令 $s = \sin^2(k\Delta x/2) \in [0,1]$：

$$\boxed{|e^{\sigma\Delta t}(c)|^2 = 1 - 4\nu(1-\nu)(1-c)\,s + 4\nu^2(1-\nu)[c^2(1-\nu) - 2c]\,s^2}$$

**各流态的局部稳定性：**

| $c$ | 流态 | $|e^{\sigma\Delta t}|^2$ | $\nu \leq 1$ 时 | 物理含义 |
|:---:|------|------|:---:|------|
| 0 | 极值点（一阶迎风） | $1 - 4\nu(1-\nu)s$ | 稳定 ✓ | 正耗散最强，所有波数衰减 |
| 1 | 光滑区（LW） | $1 + 4\nu^2(\nu^2-1)s^2$ | 稳定 ✓ | 耗散归零临界值 |
| 2 | SB 压缩区上界 | $1 + 4\nu(1-\nu)s - 16\nu^3(1-\nu)s^2$ | **可能 > 1！** | 过度反扩散，局部不稳定 |

**配图：** 见 `ppt_stability_contour.png`（由 `plot_ppt_figures.py` 生成），展示 $(c, \nu)$ 平面上的稳定性云图。横轴为限制器输出 $c = \phi(r) \in [0, 2.5]$，纵轴为 CFL 数 $\nu \in [0, 1.5]$。黑色实线为稳定性边界 $\max_s|e^{\sigma\Delta t}|^2 = 1$，红色区域为不稳定区。图中标注了 $c=0$（迎风）、$c=1$（LW）、$c=2$（SB 上界）三个关键点。

**关键发现：$c = 2$ 在 $\nu = 0.5$ 时即不稳定**（取 $s = 0.5$：$|e^{\sigma\Delta t}|^2 = 1.25 > 1$）。

但完整的 SUPERBEE 格式在 $\nu \leq 1$ 时是 TVD 且数值实验确实稳定。原因：

> **非线性耦合是稳定性的必要条件：** 限制器从不全域恒为 2——在压缩区（$c \to 2$）的相邻网格必定有 $c \to 0$ 的耗散网格作为"缓冲"，非线性格点间耦合保持全局 TVD 稳定。

**Godunov 定理在频域的微观展现：**

| | 全局冻结 $\phi = c$ | 实际限制器格式 |
|---|:---:|:---:|
| 稳定性 | 部分 $(c, \nu)$ 组合不稳定 | $\forall \nu \leq 1$：TVD → 稳定 |
| 本质 | 纯线性 | 非线性耦合自适应切换 |

**讲稿要点：**
- $c=2$ 局部不稳定但全局 TVD 稳定——这是理解"非线性 = 超越 Godunov 定理"的最直观证据
- 三种格式稳定条件相同（$\nu \leq 1$），但理论基础不同：LW = 谱半径，VL/SB = Harten TVD + 非线性耦合

**布局：** 用一张简洁的示意图展示 $c$ 在 Sweby 图上不同位置的局部稳定性

**核心信息（可口头讲解，无需单独放图）：**

- $c = 0$（一阶迎风）：强耗散稳定——正耗散系数 $\frac{a\Delta x}{2}(1-\nu) > 0$

- $c = 2$（SB 上界）：过度反扩散——**作为独立线性格式即使在 $\nu \leq 1$ 时也不稳定**
- 实际限制器格式 = 各 $c$ 分支的非线性耦合 → 全局 TVD 约束 → Sweby 区域内"安全游走"

### 第 12 页：Task 4 — 数值稳定性 · 数值验证

**布局：** 左侧放 `stability_l1.png`，右侧表格 + 三阶段分析

**方法：** 光滑初值 $\sin(\pi x)$，$N=80$，$t=10$，5 个 CFL 值 (0.5, 0.8, 1.0, 1.05, 1.1)。用 $L_1 = \frac{1}{N}\sum|u_i|$ 跟踪稳定性（精确解 $L_1^{\text{exact}} = 2/\pi \approx 0.637$，守恒）。

| CFL | LW | van Leer | SUPERBEE |
|:---:|:---:|:---:|:---:|
| 0.50 | ✓ | ✓ | ✓ |
| 0.80 | ✓ | ✓ | ✓ |
| 1.00 | ✓ | ✓ | ✓ |
| 1.05 | ✗ t≈8.7 | ✗ t≈9.0 | ✗ t≈8.8 |
| 1.10 | ✗ t≈4.9 | ✗ t≈4.9 | ✗ t≈4.8 |

**超临界 CFL 的三阶段失稳过程：**

$$\text{阶段一（视稳定）} \;\to\; \text{阶段二（拐点偏离）} \;\to\; \text{阶段三（指数增长直线）}$$

- **阶段一：** $L_1 \approx 0.637$，舍入误差 ($\sim 10^{-16}$) 中的 $2\Delta x$ 锯齿模在暗中增长
- **阶段二：** 锯齿模从 $10^{-16}$ 增长到可察觉量级 → $L_1$ 偏离 0.637
- **阶段三：** $\log(L_1)$ 与 $t$ 成线性——指数增长。直线斜率 = 格式在超临界状态的时域有效增长率

**LW vs VL/SB 的差异：**
- LW 拐点更早（无限制器缓冲），但 VL/SB 的后期斜率更陡（限制器饱和后成为"加速器"）
- VL/SB 的"视稳定期"更长——限制器在超临界初期仍在擦除锯齿模 → 推迟了发散，与 Slide 11 的分段 von Neumann 分析呼应（$c \to 0$ 强耗散）

**$10^{10}$ 阈值的局限性（简述）：**
- 单一阈值混淆了拐点时间和斜率两个独立物理量——LW 拐点更早但 VL/SB 斜率更陡，到达 $10^{10}$ 的时间却相近
- $L_1$ 偏离 $0.637$ 超过 10%（即 $L_1 > 0.7$）时解已不可用——$10^{10}$ 给出的是完全崩溃时间，非可用性终止点
- 正确分析应区分：**(a) 偏离起始时间（拐点）** 和 **(b) 指数段斜率 $\alpha$**

**讲稿要点：**
- 所有格式 $\nu \leq 1$ 稳定，$\nu > 1$ 发散——与理论一致
- $L_1 = 2/\pi$ 参考线提供早期偏离检测，优于 $\max|u|$（后者无参考常数值）
- 分段 von Neumann 分析（Slide 11）的 $c=2$ 局部不稳定 → 解释了为什么 VL/SB 在超临界时"斜率更陡"——限制器饱和后 $c \to 2$ 的那部分网格首先失稳

### 第 13 页：Task 5 — 自适应 Lax-Wendroff 改进方案

**布局：** 左侧算法思想，右侧公式

**改进目标：** 保留 LW 在光滑区的二阶精度，同时在间断附近消除振荡、保持单调。

**核心思想——激波传感器 (Shock Sensor)：**

定义局部光滑度指标 $\theta_j$：

$$\theta_j = \frac{|u_{j+1} - 2u_j + u_{j-1}|}{|u_{j+1} - u_j| + |u_j - u_{j-1}| + \varepsilon}$$

- 分子：二阶差商 ← 在间断附近很大，在光滑区趋近 0
- 分母：一阶差商之和 ← 归一化因子

$$\theta_j \to 1 \text{（间断）} \qquad \theta_j \to 0 \text{（光滑）}$$

**自适应斜率构造：**

$$\delta_j^{\text{ALW}} = (1 - \theta_j) \cdot (u_{j+1} - u_j)$$

| 区域 | θ_j | δ_j | 等效格式 |
|------|:---:|------|:---:|
| 光滑区 | → 0 | → $u_{j+1}-u_j$ | 标准 LW（二阶） |
| 间断附近 | → 1 | → 0 | 一阶迎风（单调） |
| 过渡区 | ∈ (0,1) | 自适应插值 | 介于两者之间 |

**与 van Leer / SUPERBEE 的区别：** ALW 不需要计算 $r_j$ 和限制器函数 $\phi(r)$——传感器 $\theta_j$ 直接基于解的局部曲率判断。

**讲稿要点：**
- 传感器 $\theta_j$ 的物理直觉：间断处曲率大 → 需要耗散；光滑处曲率小 → 可用二阶
- 与 VL/SB 的限制器思路不同：ALW 是"检测-切换"，VL/SB 是"连续非线性压缩"

### 第 14 页：Task 5 — 自适应 LW · 结果

**布局：** 上方两张解对比图，下方 TV 图

**配图：** `adaptive_solution_t2.png`、`adaptive_zoom_t2.png`、`adaptive_tv.png`

**结果表：**

| 格式 | min(u) @ t=8 | max(u) @ t=8 | TV @ t=8 | 单调？ |

| SUPERBEE | 0.000 | 1.000 | 2.00 | ✓ |

**关键观察：**
- ALW 完全消除了 LW 的振荡（min=0, max=1 → 严格保界）
- TV = 2.00 恒定不变 → ALW 实现了**事实上的 TVD 行为**（虽然未使用限制器）
- 在光滑区域保留了 LW 的二阶精度（传感器 θ_j ≈ 0 → 标准 LW）

**讲稿要点：**
- ALW 用约 30 行 Fortran 代码实现了与 VL/SB 接近的效果——用传感器的简洁性换取了计算效率
- TV = 2.00 说明格式在实际运行中达到了 TVD——这是通过自适应衰减而非非线性限制器实现的

### 第 15 页：综合小结

**布局：** 五大任务的统一结论表

| Task | 核心问题 | 关键结论 |
|:---:|------|------|
| 1–2 | 三种格式对比 | LW 振荡但二阶；VL/SB TVD 保单调，SB 间断最锐 |
| 3 | 精度阶数 | LW 全局二阶 (L∞≈2)；VL/SB 仅 L₁ 二阶 (L∞≈1.1–1.4) |
| 4 | 稳定性条件 | 三者均为 ν ≤ 1；ν > 1 时指数发散，TVD 不改变稳定边界 |
| 5 | LW 改进 | ALW 传感器方案消除振荡，TV=2.0 恒定，光滑区保留二阶 |

**底部一句话结论：**

> **线性二阶格式必有振荡（Godunov 定理）。解决路径：非线性限制器（VL/SB）或自适应传感器（ALW）——核心思想都是在间断附近局部降阶耗散，在光滑区保留二阶精度。**
