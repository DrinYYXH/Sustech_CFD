## 理论基础

### 1. 控制方程与解析解

#### 1.1 一维线性对流方程

考虑如下一阶双曲型偏微分方程：

$$\frac{\partial u}{\partial t} + a\frac{\partial u}{\partial x} = 0, \quad a = 1$$

方程的**特征线**满足 $\mathrm{d}x/\mathrm{d}t = a$。沿特征线，解为常数：

$$\frac{\mathrm{d}u}{\mathrm{d}t} = \frac{\partial u}{\partial t} + \frac{\mathrm{d}x}{\mathrm{d}t}\frac{\partial u}{\partial x} = \frac{\partial u}{\partial t} + a\frac{\partial u}{\partial x} = 0$$

因此解析解为 $u(x,t) = u_0(x - at)$，即初值以速度 $a$ 沿 $x$ 方向平移，波形不变。

#### 1.2 周期边界与本问题的解析解

计算域为 $x \in [-1, 1]$，周期长度 $L = 2$。周期边界条件：

$$u(-1, t) = u(1, t), \quad \forall t$$

解析解通过周期延拓获得：波形平移 $at$ 后，落出域的部分从另一侧重新进入。对于本项目使用的方波初值：

$$u_0(x) = \begin{cases} 1.0, & -0.5 < x < 0.5 \\ 0.0, & \text{otherwise} \end{cases}$$

在 $t = 2.0$ 时波平移 $at = 2.0$，恰好走完一个周期回到原位；$t = 8.0$ 时走完四个周期。因此在这两个输出时刻，**解析解与初值完全重合**，为数值解的比较提供了极大的便利。

#### 1.3 间断初值的特殊性

方波初值在 $x = \pm 0.5$ 处存在跳跃间断。尽管解析解形式上只需平移初值，但间断给数值求解带来了根本性挑战：

- 间断处解不可导，标准 Taylor 展开失效
- 线性格式（如 Lax-Wendroff）在间断附近产生非物理的色散振荡（Gibbs 现象）
- 高阶格式的精度优势在间断处会丧失

这构成了引入**非线性限制器**和**TVD 理论**的根本动机。

### 2. 一阶迎风格式与数值耗散

#### 2.1 格式推导

考虑 $a > 0$（信息从左侧传播到右侧），使用后向差分逼近空间导数：

$$\frac{u_j^{n+1} - u_j^n}{\Delta t} + a\frac{u_j^n - u_{j-1}^n}{\Delta x} = 0$$

整理得**一阶迎风格式**（First-Order Upwind, FOU）：

$$u_j^{n+1} = u_j^n - \nu(u_j^n - u_{j-1}^n), \quad \nu = \frac{a\Delta t}{\Delta x}$$

#### 2.2 修正方程与数值耗散

将 $u_j^{n+1}$ 和 $u_{j-1}^n$ 在 $(x_j, t^n)$ 处做 Taylor 展开，代入迎风格式，消去满足 PDE 的 $\partial u/\partial t$ 项，得到**修正方程**（Modified Equation）：

$$\frac{\partial u}{\partial t} + a\frac{\partial u}{\partial x} = \underbrace{\frac{a\Delta x}{2}(1-\nu)}_{\text{数值耗散系数}} \frac{\partial^2 u}{\partial x^2} + \mathcal{O}(\Delta x^2, \Delta t^2)$$

修正方程右侧的二阶导数项即为**数值耗散**（numerical diffusion），其系数 $\alpha_{\text{num}} = a\Delta x(1-\nu)/2$。

**关键性质：**
- 当 $\nu \leq 1$ 时，$\alpha_{\text{num}} \geq 0$，耗散为正（使解趋于光滑）
- 正耗散使间断被抹平（间断宽度以 $\sim\sqrt{\alpha_{\text{num}} t}$ 增长）
- 该格式为**一阶精度**（$\mathcal{O}(\Delta x, \Delta t)$），无振荡但过度耗散

### 3. Lax-Wendroff 格式与色散误差

#### 3.1 格式推导

Lax-Wendroff 格式的基本思路：利用 PDE 本身将时间导数转化为空间导数，通过 Taylor 展开保留二阶项。

对 $u(x, t+\Delta t)$ 做 Taylor 展开至二阶：

$$u(t+\Delta t) = u + \Delta t\frac{\partial u}{\partial t} + \frac{\Delta t^2}{2}\frac{\partial^2 u}{\partial t^2} + \mathcal{O}(\Delta t^3)$$

利用 PDE：$\partial u/\partial t = -a\partial u/\partial x$，以及 $\partial^2 u/\partial t^2 = a^2\partial^2 u/\partial x^2$，得：

$$u(t+\Delta t) = u - a\Delta t\frac{\partial u}{\partial x} + \frac{a^2\Delta t^2}{2}\frac{\partial^2 u}{\partial x^2} + \mathcal{O}(\Delta t^3)$$

中心差分离散空间导数，得到经典的**两步 Lax-Wendroff 格式**：

$$u_j^{n+1} = u_j^n - \frac{\nu}{2}(u_{j+1}^n - u_{j-1}^n) + \frac{\nu^2}{2}(u_{j+1}^n - 2u_j^n + u_{j-1}^n)$$

#### 3.2 修正方程与色散误差

Taylor 展开至三阶得 LW 的修正方程：

$$\frac{\partial u}{\partial t} + a\frac{\partial u}{\partial x} = \underbrace{\frac{a\Delta x^2}{6}(1-\nu^2)\frac{\partial^3 u}{\partial x^3}}_{\text{色散误差项}} + \mathcal{O}(\Delta x^3)$$

**关键性质：**
- LW 格式为**二阶精度**（$\mathcal{O}(\Delta x^2, \Delta t^2)$），空间和时间均二阶
- 无偶数阶耗散项（$\partial^2/\partial x^2$ 项被抵消，这是 LW 设计的精妙之处）
- 主导误差为**三阶色散项**：$\partial^3 u/\partial x^3$ 造成不同波数分量的相速度不同（色散），表现为 $2\Delta x$ 波长的高频振荡
- 在间断附近，色散误差被显著放大，产生**非物理的 Gibbs 振荡**

#### 3.3 LW 方案适定性与色散振荡机制

理解 LW 振荡的物理解释：间断可以看作所有频率 Fourier 分量的叠加。不同波数的 Fourier 模在 LW 格式下的传播速度（数值相速度）不同——高频分量传播过慢或过快。这种色散导致高频分量在间断前后堆积，形成振荡。一阶迎风格式的耗散恰好抑制了这些高频分量（代价是抹平间断），而 LW 因缺乏耗散机制无法抑制。

**这正是本项目需要引入限制器的根本原因**：保留 LW 的二阶精度，同时用非线性方式在间断附近施加耗散。

### 4. MUSCL 重构框架

#### 4.1 从 Godunov 格式到 MUSCL

Godunov 定理指出：**单调的线性格式至多为一阶精度**。这意味着要获得高于一阶的精度同时保持单调性，必须使用**非线性格式**。

MUSCL（Monotonic Upstream-centered Scheme for Conservation Laws）由 van Leer（1979）提出，核心思想是：
1. 在每个单元内构造**分段线性重构**
2. 使用**限制器**（limiter）约束重构斜率，防止产生新的极值

#### 4.2 单元线性重构

在单元 $j$ 内，假设解为线性分布：

$$u_j(x) = u_j^n + \sigma_j^n (x - x_j), \quad x \in [x_{j-1/2}, x_{j+1/2}]$$

其中 $\sigma_j^n$ 是单元 $j$ 内的斜率。通过该重构可获得单元界面处的左右状态值：

$$u_{j+1/2}^L = u_j + \frac{\Delta x}{2}\sigma_j, \qquad u_{j-1/2}^R = u_j - \frac{\Delta x}{2}\sigma_j$$

#### 4.3 统一 MUSCL 时间推进公式

将分段线性重构代入迎风 Godunov 型通量并做积分平均，得到本项目使用的**统一 MUSCL 格式**（逐单元的公式 3）：

$$\boxed{u_j^{n+1} = u_j^n - \nu\big(u_j^n - u_{j-1}^n\big) - \frac{\nu(1-\nu)}{2}\big(\delta_j - \delta_{j-1}\big)}$$

其中 $\nu = a\Delta t/\Delta x$ 为 CFL 数，$\delta_j \equiv \sigma_j \Delta x$ 为单元 $j$ 的斜率与网格间距的乘积。

**公式结构解析：**
- 第一项 $u_j^n$：当前值
- 第二项 $-\nu(u_j^n - u_{j-1}^n)$：一阶迎风通量（保证基本输送）
- 第三项 $-\frac{\nu(1-\nu)}{2}(\delta_j - \delta_{j-1})$：**反扩散修正项**（提高精度至二阶）

反扩散修正项的作用是在迎风基础上补偿数值耗散——当限制器允许时（光滑区），$\delta_j$ 取满值恢复二阶精度；当限制器起作用时（间断附近），$\delta_j$ 被压缩以抑制振荡。

#### 4.4 斜率选择与格式的关系

统一框架中，$\delta_j$ 的选择决定了格式性质：

- **下风斜率** $\delta_j = u_{j+1} - u_j$ → 代入后得到等价于经典 Lax-Wendroff 的格式
- **带限制器的斜率** $\delta_j = \phi(r_j) \cdot (u_{j+1} - u_j)$ → 非线性 TVD 格式

斜率比 $r_j$ 的定义为前后差商的比值：

$$r_j = \frac{u_j - u_{j-1}}{u_{j+1} - u_j}$$

当 $r_j \approx 1$（解局部线性）时，限制器 $\phi \approx 1$，恢复全二阶精度；当 $r_j$ 偏离 1（存在间断或极值）时，$\phi$ 减小以施加耗散。

### 5. TVD 理论与限制器设计

#### 5.1 全变差（Total Variation）

一维离散函数的全变差定义为：

$$\text{TV}(u^n) = \sum_{i} |u_{i+1}^n - u_i^n|$$

在周期域上求和含卷绕项 $|u_1 - u_N|$。TV 衡量了解的**总振荡量**——对单调初值，解析解的 TV 在时间演化中保持不变（对于线性对流）。

#### 5.2 TVD 条件与 Harten 定理

一个格式称为**全变差不增**（Total Variation Diminishing, TVD），如果对所有 $n$：

$$\text{TV}(u^{n+1}) \leq \text{TV}(u^n)$$

**Harten 定理（1983）：** 若格式可写为以下形式：

$$u_j^{n+1} = u_j^n - C_{j-1/2}(u_j^n - u_{j-1}^n) + D_{j+1/2}(u_{j+1}^n - u_j^n)$$

且系数满足 $C_{j-1/2} \geq 0$，$D_{j+1/2} \geq 0$，$C_{j-1/2} + D_{j+1/2} \leq 1$，则该格式为 TVD。

**TVD 格式的优势：**
- 自动满足单调性（无新增局部极值）
- 不会产生非物理振荡
- $L_\infty$ 稳定性：$\max|u^n| \leq \max|u^0|$（对于线性对流）

#### 5.3 限制器的约束条件

为使统一 MUSCL 格式成为 TVD，限制器函数 $\phi(r)$ 必须满足以下约束（对 $\nu \leq 1$）：

$$0 \leq \phi(r) \leq 2, \qquad 0 \leq \frac{\phi(r)}{r} \leq 2$$

此外，TVD 限制器还必须满足**对称性条件**——格式不应因使用前方差商或后方差商而产生不对等的行为：

$$\frac{\phi(r)}{r} = \phi\!\left(\frac{1}{r}\right)$$

对称性的物理意义：如果将"前方"和"后方"的角色互换（$r \to 1/r$），限制器对斜率的压缩程度应当一致。这个条件意味着：**只需定义 $r \in [0, 1]$ 内的 $\phi(r)$，$r > 1$ 的部分由对称性自动生成。** 这就是为什么很多教材和老师只画 $0 \leq r \leq 1$ 的 Sweby 图——另一半是对称镜像。

这定义了著名的 **Sweby 图中的 TVD 区域**。

### 6. 限制器函数与 Sweby 图

**核心概念：$r$ 是比值，$\phi(r)$ 是系数，最终斜率是差值。**

整个限制器框架涉及三个量，容易混淆：

| 量 | 定义 | 量纲 | 含义 |
|----|------|:---:|------|
| $r_j$ | $\dfrac{u_j - u_{j-1}}{u_{j+1} - u_j}$ | **无量纲（比值）** | 前后差商的比值，衡量解的局部光滑度 |
| $\phi(r_j)$ | 限制器函数 | **无量纲（系数）** | 根据 $r_j$ 算出的压缩因子 |
| $\delta_j$ | $\phi(r_j) \cdot (u_{j+1} - u_j)$ | **$u$ 的量纲（差值）** | 最终用于 MUSCL 格式的斜率 |

所以 Sweby 图讨论的 $\phi(r)$ — 横轴 $r$ 是比值，纵轴 $\phi$ 是系数——都是无量纲的。**实际进入公式的是差值** $\delta_j = \phi(r_j) \times$ 前方差商。

举例：若某网格处 $u_{j-1}=0$，$u_j=0.2$，$u_{j+1}=1.0$，则：
- $r_j = (0.2-0)/(1.0-0.2) = 0.25$
- 查 SUPERBEE 分段：$0 < 0.25 \leq 0.5$，所以 $\phi = 2 \times 0.25 = 0.5$
- 最终斜率 $\delta_j = 0.5 \times (1.0 - 0.2) = 0.4$（这是一个差值，和 $u$ 同单位）

---

#### 6.1 常见限制器

本项目使用的三种方案对应不同的 $\phi(r)$ 选择：

**Lax-Wendroff（无限限制器 / 下风斜率）：**

$$\phi_{\text{LW}}(r) = 1$$

- 对所有 $r$ 都取满下风斜率
- 二阶精度，但非 TVD（超出 Sweby 约束）

**van Leer 限制器（形式 2）：**

$$\phi_{\text{VL}}(r) = \frac{r + |r|}{1 + |r|} = \begin{cases} 0, & r \leq 0 \\ \dfrac{2r}{1+r}, & r > 0 \end{cases}$$

- 光滑可微（在 Sweby 图中是介于上界 $\phi=2$ 和 $\phi=2r$ 之间的光滑曲线）
- 对称限制，收敛行为良好
- 在间断处给出适中的压缩度

**SUPERBEE 限制器（Roe, 1985）：**

SUPERBEE 有两种完全等价的写法：

**(1) 代码形式 — `max min` 写法：**

$$\phi_{\text{SB}}(r) = \max\bigl(0,\; \min(2r, 1),\; \min(r, 2)\bigr)$$

**(2) 分段形式 — 更直观的理解：**

$$\phi_{\text{SB}}(r) = \begin{cases} 0, & r \leq 0 \\[4pt] 2r, & 0 < r \leq \dfrac{1}{2} \\[8pt] 1, & \dfrac{1}{2} < r \leq 1 \\[8pt] r, & 1 < r \leq 2 \\[4pt] 2, & r > 2 \end{cases}$$

**(3) 两种形式的等价性证明：**

`min(2r, 1)` 是 $2r$ 和 $1$ 两条函数中取较小者，`min(r, 2)` 是 $r$ 和 $2$ 中取较小者，最外层 `max(0, ..., ...)` 是三个值中取最大者。逐区间检验：

| $r$ 区间 | $\min(2r, 1)$ 的值 | $\min(r, 2)$ 的值 | 三个数取 max | 分段结果 |
|:---:|:---:|:---:|:---:|:---:|
| $r \leq 0$ | $2r$（$\leq 0$） | $r$（$\leq 0$） | $\max(0, \leq 0, \leq 0) = 0$ | **0** |
| $0 < r \leq \frac{1}{2}$ | $2r$（$< 1$） | $r$（$\leq 2$） | $\max(0, 2r, r) = 2r$ | **2r** |
| $\frac{1}{2} < r \leq 1$ | $1$（$2r$ 已超过 $1$） | $r$（$\leq 2$） | $\max(0, 1, r) = 1$ | **1** |
| $1 < r \leq 2$ | $1$（$2r$ 已超过 $1$） | $r$（$\leq 2$） | $\max(0, 1, r) = r$ | **r** |
| $r > 2$ | $1$（$2r > 1$） | $2$（$r$ 已超过 $2$） | $\max(0, 1, 2) = 2$ | **2** |

五个区间全部吻合，代码形式与分段形式**严格等价**。

**(4) 图形理解：**

想象三条参考线：
- 第一条：$\phi = 2r$（斜率 2 的直线），但不超过 $\phi = 1$
- 第二条：$\phi = r$（斜率 1 的直线），但不超过 $\phi = 2$
- 第三条：$\phi = 0$（横轴）

三个函数画在 Sweby 图上，每一步的 $\phi(r)$ 取三者的**上包络**（最大值），得到的折线恰好就是分段形式描述的 5 段折线。

**(5) SUPERBEE 的物理本质：**

- 取 Sweby 图中 TVD 区域的上包络（上下界中取最大允许值）
- 这是**所有 TVD 限制器中压缩度最高、间断分辨率最锐利**的方案
- 代价是可能在光滑极值附近产生轻微"阶梯效应"（staircasing），且 $\phi(r)$ 在 $r = 1/2$ 和 $r = 2$ 处不可导
- 与 van Leer 对比：van Leer 取 TVD 区域的中间光滑曲线，SUPERBEE 取上边界

#### 6.2 r ≤ 0 时的处理

当 $r_j \leq 0$ 时，意味着 $u_{j+1}-u_j$ 与 $u_j-u_{j-1}$ 符号相反，此时单元 $j$ 处于**局部极值点**。TVD 限制器此时必须返回 $\phi = 0$（即退化为迎风），否则会产生新的极值——这是上帝定理的直接推论。

#### 6.3 r > 1 的物理含义与限制器的对称性

**为什么 $r$ 会大于 1？**

$r_j = (u_j - u_{j-1})/(u_{j+1} - u_j)$，分母是前方差商，分子是后方差商。当后方差商大于前方差商时，$r > 1$。这在以下场景中频繁出现：

- **光滑峰值下游**：例如 $\sin$ 波在过了波峰后，梯度绝对值在减小。前一步的下降量（后方）大于下一步的下降量（前方），$r > 1$。
- **激波/间断后的松弛区**：经过间断后，解从突变状态逐渐"平稳"回到常值。后方的大梯度（跨越间断）与前方的小梯度（接近常值）形成 $r > 1$。
- **方波平顶右边界附近**：在 $u = 1$ 到 $u = 0$ 的下跳沿，紧邻跳沿左侧的网格，后方（平台区）变化接近 0，前方（下降区）变化为负值 → $r \leq 0$。但距跳沿 2-3 格处，梯度在缓和，$r > 1$ 出现。

简单说：$r < 1$ 是"梯度在加剧"（向间断逼近），$r = 1$ 是"梯度不变"（局部线性），$r > 1$ 是"梯度在缓和"（离开间断或从极值恢复）。

**限制器的对称性（Sweby 对称条件）：**

TVD 限制器必须满足一个对称性要求——格式不应对前方和后方有偏袒：

$$\frac{\phi(r)}{r} = \phi\!\left(\frac{1}{r}\right)$$

这个对称条件意味着：
- 知道 $r \in [0, 1]$ 的 $\phi(r)$ 后，$r \in [1, \infty)$ 的 $\phi(r)$ 由对称性自动确定
- 这就是为什么很多教材和老师只展示 $0 \leq r \leq 1$ 的那一段——另外一半是镜像生成的

**SUPERBEE 中 r > 1 部分的推导：**

利用对称性，从 $r$ 的已知左侧部分推导右侧部分：

| $r$ 区间 | 对称映射 | 推导 | $\phi(r)$ |
|:---:|:---:|------|:---:|
| $1 < r \leq 2$ | $\frac{1}{r} \in [\frac{1}{2}, 1]$ | $\phi(r) = r \cdot \phi(1/r) = r \cdot 1$ | **$r$** |
| $r > 2$ | $\frac{1}{r} \in (0, \frac{1}{2})$ | $\phi(r) = r \cdot \phi(1/r) = r \cdot 2(1/r)$ | **2** |

这就是 SUPERBEE 分段形式中后两段的来源：

$$\phi(r) = \begin{cases} r, & 1 < r \leq 2 \quad\text{（对称性的必然结果）}\\ 2, & r > 2 \quad\text{（对称性的必然结果）} \end{cases}$$

**以方波为例看 r > 1 的实际出现：**

取 $t=2.0$ 时 van Leer 解在左间断附近的一段数据：

| 位置 | $u$ 值 | $r_j$ | 所属区间 | $\phi_{\text{SB}}$ |
|------|--------|:---:|:---:|:---:|
| 间断左侧远处 | 0.000 | — | $r \approx 0$ | 0 |
| 进入上升沿 | 0.084 | 0.26 | $0 < r \leq 0.5$ | 0.52 |
| 上升沿中段 | 0.425 | 0.82 | $0.5 < r \leq 1$ | 1.00 |
| **离开上升沿** | **0.895** | **1.85** | **$1 < r \leq 2$** | **1.85** |
| **平台过渡** | **0.982** | **4.87** | **$r > 2$** | **2.00** |

可以看到 $r$ 从 0 逐渐增大到远超 1——这是一个真实物理过程的自然结果。没有 $r > 1$ 的定义，SUPERBEE 在"离开间断进入平滑区"的过渡地带将失效。

#### 6.4 限制器比较

| 限制器 | $\phi$ 光滑性 | 压缩度 | 间断分辨率 | TVD |
|--------|:---:|:---:|:---:|:---:|
| 无限制（LW） | 是 | — | 有振荡 | ✗ |
| van Leer | 是 | 中等 | ~7 网格 | ✓ |
| SUPERBEE | 否 | 最强 | ~5 网格 | ✓ |

### 7. von Neumann 稳定性分析

#### 7.1 方法概述

von Neumann（Fourier）稳定性分析假设解可分解为 Fourier 模：

$$u_j^n = G^n \cdot e^{ikx_j}$$

其中 $G$ 为**放大因子**（amplification factor）。格式稳定的充要条件为：

$$|G(k)| \leq 1, \quad \forall k \in [0, \pi/\Delta x]$$

#### 7.2 Lax-Wendroff 格式的 von Neumann 分析

将 Fourier 模代入 LW 格式：

$$u_j^{n+1} = u_j^n - \frac{\nu}{2}(u_{j+1}^n - u_{j-1}^n) + \frac{\nu^2}{2}(u_{j+1}^n - 2u_j^n + u_{j-1}^n)$$

利用 $u_{j\pm1}^n = G^n e^{ik(x_j \pm \Delta x)}$，提取放大因子：

$$G(k) = 1 - \nu^2(1 - \cos k\Delta x) - i\nu\sin k\Delta x$$

其模的平方为：

$$\boxed{|G(k)|^2 = 1 - 4\nu^2(1-\nu^2)\sin^4\left(\frac{k\Delta x}{2}\right)}$$

**稳定性条件推导：** 要求 $|G|^2 \leq 1$ 对所有 $k$ 成立：

$$1 - 4\nu^2(1-\nu^2)\sin^4\left(\frac{k\Delta x}{2}\right) \leq 1$$

$$\Rightarrow \nu^2(1-\nu^2) \geq 0$$

$$\Rightarrow \boxed{\nu \leq 1}$$

这就是 Lax-Wendroff 格式的 CFL 稳定性条件。当 $\nu = 1$ 时，$|G| = 1$（对所有 $k$），格式无耗散无放大——此为单位 CFL 的精确输送性质。

#### 7.3 物理意义

von Neumann 分析本质上是检查：是否存在某个 Fourier 模在每一步时间推进后被放大。若 $|G| > 1$，该模将指数增长最终覆盖真实解。对于 LW，$\nu > 1$ 时高频模（$k\Delta x \approx \pi$）最先生长，表现为 $2\Delta x$ 锯齿振荡。

---

### 8. TVD 稳定性与 Harten 定理

#### 8.1 非线性稳定性

对于 van Leer 和 SUPERBEE 这类非线性格式，von Neumann 分析不直接适用（因为限制器引入了与 $u$ 相关的非线性系数）。取而代之的是 **TVD 稳定性理论**。

#### 8.2 统一 MUSCL 框架的 TVD 条件

将统一 MUSCL 格式改写为 Harten 形式：

$$u_j^{n+1} = u_j^n - \nu\big(u_j^n - u_{j-1}^n\big) - \frac{\nu(1-\nu)}{2}\big[\phi(r_j)(u_{j+1}-u_j) - \phi(r_{j-1})(u_j-u_{j-1})\big]$$

整理为标准三对角形式。当限制器 $\phi(r)$ 满足 Sweby 约束：

$$0 \leq \phi(r) \leq 2, \quad 0 \leq \frac{\phi(r)}{r} \leq 2$$

且 CFL 条件 $\nu \leq 1$ 成立时，可证明系数满足 Harten 定理的非负性条件，从而方案为 TVD。

**关键结论：** 非线性限制器格式在 $\nu \leq 1$ 时既是 TVD（保单调、无振荡）也是稳定的。CFL 条件与线性稳定性分析结果一致，但保证方式不同——线性格式靠耗散/无放大，非线性格式靠 TVD 性质直接约束了解的变化范围。

#### 8.3 稳定性条件的数值奇异性

在 $\nu = 1$ 时，统一公式中的反扩散项系数 $\nu(1-\nu)/2 = 0$，格式退化为单纯的一阶迎风（此时恰好精确输送，因 $u_j^{n+1} = u_j^n - (u_j^n - u_{j-1}^n) = u_{j-1}^n$）。$\nu$ 略大于 1 时，反扩散系数变号，格式立即失稳。

### 9. 精度阶数分析方法

确定格式精度有两种互补方法：**Taylor 展开（理论推导）**和**网格加密（数值验证）**。Taylor 展开从差分方程本身直接导出截断误差的阶数，不需要运行任何代码；网格加密则通过实测误差的衰减率来验证理论预测。

#### 9.1 Taylor 展开法 — 从理论上确定精度阶数

截断误差 $\tau$ 定义为将精确解代入差分格式后的余项：

$$\tau_j^n = \frac{u(x_j, t^{n+1}) - u(x_j, t^n)}{\Delta t} + a\frac{u(x_j, t^n) - u(x_{j-1}, t^n)}{\Delta x} + \frac{a(\Delta x - a\Delta t)}{2\Delta x}\big(\sigma_j^n - \sigma_{j-1}^n\big)$$

将 $u(x_j, t^{n+1})$、$u(x_{j-1}, t^n)$ 等在 $(x_j, t^n)$ 处做 Taylor 展开，利用 PDE $u_t + au_x = 0$ 消去时间导数，得到截断误差主项。**$\tau$ 中 $\Delta x$ 的最低幂次即为格式的理论精度阶数。**

---

**（1）Lax-Wendroff 格式（$\delta_j = u_{j+1} - u_j$）**

将 $\delta_j = u_{j+1} - u_j$ 代入统一 MUSCL 公式：

$$u_j^{n+1} = u_j - \nu(u_j - u_{j-1}) - \frac{\nu(1-\nu)}{2}\big[(u_{j+1}-u_j) - (u_j-u_{j-1})\big]$$

$$= u_j - \frac{\nu}{2}(u_{j+1} - u_{j-1}) + \frac{\nu^2}{2}(u_{j+1} - 2u_j + u_{j-1})$$

这正是经典的 Lax-Wendroff 两步等价形式。分别展开各差商：

$$\frac{u_{j+1} - u_{j-1}}{2\Delta x} = u_x + \frac{\Delta x^2}{6}u_{xxx} + \mathcal{O}(\Delta x^4)$$

$$\frac{u_{j+1} - 2u_j + u_{j-1}}{\Delta x^2} = u_{xx} + \frac{\Delta x^2}{12}u_{xxxx} + \mathcal{O}(\Delta x^4)$$

代入格式并减去精确解 $u(x_j, t^n+\Delta t)$ 的 Taylor 展开：

$$u(x_j, t^n+\Delta t) = u + \Delta t u_t + \frac{\Delta t^2}{2}u_{tt} + \frac{\Delta t^3}{6}u_{ttt} + \mathcal{O}(\Delta t^4)$$
$$= u - a\Delta t u_x + \frac{a^2\Delta t^2}{2}u_{xx} - \frac{a^3\Delta t^3}{6}u_{xxx} + \mathcal{O}(\Delta t^4, \Delta x^4)$$

整理得截断误差：

$$\tau_{\text{LW}} = -\frac{a\Delta x^2}{6}(1-\nu^2)u_{xxx} + \mathcal{O}(\Delta x^3)$$

**结论：** 主导截断误差为 $\mathcal{O}(\Delta x^2)$ 乘以三阶空间导数（色散项）。LW 格式在空间和时间上均达到**二阶精度**。

---

**（2）van Leer 格式（$\delta_j = \phi_{\text{VL}}(r_j)(u_{j+1}-u_j)$）**

在光滑单调区域，$u_{j-1}$、$u_j$、$u_{j+1}$ 近似线性分布，斜率比 $r_j \approx 1$。此时：

$$\phi_{\text{VL}}(1) = \frac{1+|1|}{1+|1|} = 1$$

因此 $\delta_j \approx u_{j+1} - u_j$，格式退化为 LW，截断误差同 LW：$\tau \sim \mathcal{O}(\Delta x^2)$。

在**局部极值点**（$r_j \leq 0$），van Leer 限制器返回 $\phi_{\text{VL}} = 0$，$\delta_j = 0$。此时统一公式退化为：

$$u_j^{n+1} = u_j - \nu(u_j - u_{j-1})$$

这是一阶迎风格式。其截断误差（参见第 2 节）：

$$\tau_{\text{FOU}} = \frac{a\Delta x}{2}(1-\nu)u_{xx} + \mathcal{O}(\Delta x^2)$$

**结论：** van Leer 格式在光滑单调区为**二阶精度**，在极值点局部降为**一阶精度**。全局 $L_1$ 范数受控于光滑区（占据绝大多数网格），仍表现为接近二阶；$L_\infty$ 范数被极值点的一阶局部误差主导，表现为更低阶。

---

**（3）SUPERBEE 格式（$\delta_j = \phi_{\text{SB}}(r_j)(u_{j+1}-u_j)$）**

同理，在光滑单调区 $r_j \approx 1$：

$$\phi_{\text{SB}}(1) = \max\big(0, \min(2, 1), \min(1, 2)\big) = 1$$

$\delta_j \approx u_{j+1} - u_j$，格式 → LW，截断误差 $\mathcal{O}(\Delta x^2)$。

在极值点 $\phi_{\text{SB}} = 0$，退化为迎风，截断误差 $\mathcal{O}(\Delta x)$。

由于 SUPERBEE 是最激进的限制器（取 Sweby 图 TVD 区的上包络），它在非极值但梯度变化较大处也倾向于返回较小的 $\phi$ 值（尽管仍在 Sweby 边界内），导致 $L_\infty$ 误差略大于 van Leer。

**结论：** SUPERBEE 在光滑单调区为**二阶精度**，极值点处一阶。$L_\infty$ 阶数比 van Leer 更低（极限器更强）。

---

**三种格式理论精度的统一表述：**

| 格式 | 光滑单调区截断误差主项 | 理论阶 | 极值点行为 |
|------|----------------------|:---:|------|
| LW | $-\frac{a\Delta x^2}{6}(1-\nu^2)u_{xxx}$ | **2** | 仍为二阶（但产生振荡） |
| van Leer | 同 LW（$\phi\to1$） | **2** | 降为一阶（$\phi\to0$） |
| SUPERBEE | 同 LW（$\phi\to1$） | **2** | 降为一阶（$\phi\to0$） |

---

#### 9.2 网格加密法 — 数值验证

精度阶数的数值验证通过**网格加密**（Grid Refinement Study）完成。核心思想：在越来越密的网格上求解同一问题，测量误差随 $\Delta x$ 的衰减速率。

若误差 $E(\Delta x) \approx C \cdot \Delta x^p$，则：

$$p \approx \frac{\log(E_1/E_2)}{\log(\Delta x_1/\Delta x_2)}$$

以 $\log(\Delta x)$ 为横轴、$\log(E)$ 为纵轴做图，斜率即为实测收敛阶 $p$。

本项目使用 $N = 20, 40, 80, 160, 320$（$\Delta x$ 逐次减半），CFL 固定 0.8，计算至 $t=1.0$。

#### 9.3 光滑初值的必要性

Taylor 展开给出的理论精度假设解足够光滑（可 Taylor 展开至所需阶数）。在含间断的问题中，解本身不可导，格式的实际收敛率受限于解的**正则性**：

$$\|\text{error}\| \leq C \cdot \Delta x^{\min(s, p)}$$

其中 $s$ 为解在对应范数下的光滑度，$p$ 为格式的标称精度。方波初值 $s \approx 0.5$，因此所有格式的实测收敛阶都将降至约 $\mathcal{O}(\Delta x^{0.5\sim0.8})$，无法体现格式的真实精度。

使用 $\sin(\pi x)$ 初值（$C^\infty$）消除了正则性限制，使格式的**标称精度**得以在数值上验证。**实测收敛阶应当与 Taylor 展开的理论预测一致。**

#### 9.4 误差范数

常用两种误差范数：

$$L_1 = \frac{1}{N}\sum_{i=1}^{N} |u_i - u_i^{\text{exact}}|, \qquad L_\infty = \max_{1 \leq i \leq N} |u_i - u_i^{\text{exact}}|$$

- $L_1$：全局平均误差，对局部振荡不敏感，反映格式在大多数网格上的精度
- $L_\infty$：最大点误差，反映最差情况，对限制器在极值点的降阶非常敏感

**对于限制器格式**：即使 Taylor 展开预测光滑区为二阶，$L_\infty$ 范数会被极值点的一阶局部误差主导，表现为约 $\mathcal{O}(\Delta x^{1.0\sim1.4})$。$L_1$ 仍接近二阶，因为极值点仅占全体网格的极小比例。

#### 9.5 代码设计细节

- CFL 固定 0.8，确保所有网格使用相同的 $\nu$（$\nu$ 影响截断误差系数）
- $t_{\text{end}} = 1.0$，避免波走完整周期导致误差抵消
- 实际时间 `t_actual = nint(t_end/dt) * dt`，确保解析解在同一时刻评估
- 使用动态内存分配（`allocate`/`deallocate`）适应 $N=20 \sim 320$

### 10. 激波传感器理论

#### 10.1 设计动机

Lax-Wendroff 的限制器直接钳制斜率，对所有网格施加非线性处理。自适应 LW 采用不同的思路：**仅在需要的位置施加耗散，其余位置保留 LW 的满二阶精度**。

#### 10.2 Jameson 型激波传感器

传感器的核心是利用二阶差商与一阶差商的比值检测间断：

$$\theta_j = \frac{|u_{j+1} - 2u_j + u_{j-1}|}{|u_{j+1} - u_j| + |u_j - u_{j-1}| + \varepsilon}$$

**物理诠释（以 $u$ 类比压力）：**

- **光滑区域**：$u$ 近似线性，$u_{j+1} - 2u_j + u_{j-1} \approx \Delta x^2 \cdot u'' \approx 0$ → 分子约零，$\theta_j \approx 0$
- **间断附近**：$u$ 突变，二阶差商 $\approx$ 一阶差商量级 → 分子分母同量级，$\theta_j \approx 1$
- **极值点**：一阶差商小，二阶差商大 → $\theta_j \to 1$，自动施加耗散（这正是 TVD 要求在极值点降阶的体现）

分母中加入 $\varepsilon = 10^{-12}$ 防止除零。分子取 $|u_{j+1} - 2u_j + u_{j-1}|$ 恰为离散二阶导数绝对值（乘以 $\Delta x^2$），是对局部曲率的度量。

#### 10.3 自适应斜率

$$\delta_j^{\text{ALW}} = (1 - \theta_j) \cdot (u_{j+1} - u_j)$$

- $\theta_j \approx 0$（光滑）→ $\delta_j^{\text{ALW}} \approx u_{j+1} - u_j$（标准 LW 下风斜率）
- $\theta_j \approx 1$（间断）→ $\delta_j^{\text{ALW}} \approx 0$（退化为迎风，正耗散抑制振荡）

$\theta_j$ 的平滑过渡实现了光滑区的高精度与间断区的单调性之间的**自动连续切换**，无需显式区分区域。这本质上是一种**非线性耗散机制**——耗散系数随局部光滑度自适应调节。

#### 10.4 与限制器的比较

| 特性 | 限制器（VL/SB） | 激波传感器（ALW） |
|------|:---:|:---:|
| 是否修改斜率 | 是，基于 $\phi(r_j)$ | 是，基于 $(1-\theta_j)$ |
| 光滑区行为 | $\phi \approx 1$，接近 LW | $\theta \approx 0$，= LW |
| 间断行为 | $\phi$ 减小以压缩斜率 | $\theta \to 1$，退化为迎风 |
| 过渡 | 部分区域限制器过度压缩 | 传感器给出更平滑的过渡 |
| 单调性保证 | TVD（严格） | 限制后 TV ≈ 常数（实践中 TVD）|

**作用：** 使用光滑初值 $u(x,0)=\sin(\pi x)$，在 5 组不同网格（N=20, 40, 80, 160, 320）上运行三种格式，计算 $L_1$ 和 $L_\infty$ 误差范数，通过网格加密法**数值验证**各格式的精度阶数。

**与 Taylor 展开理论分析的关系：**
精度阶数可从两个途径确定——**Taylor 展开（理论推导）**和**网格加密（数值验证）**。Taylor 展开直接对差分格式做截断误差分析（详见理论基础第 9.1 节），从理论上给出 LW 为 $\mathcal{O}(\Delta x^2)$、VL 和 SB 在光滑区 $\mathcal{O}(\Delta x^2)$ 但在极值点局部降为一阶。网格加密法通过实测误差衰减率来**验证**这些理论预测。

**为什么用光滑初值？**
方波初值本身不可导，解也不光滑，所有格式在间断处的收敛率都将退化到 $\mathcal{O}(\Delta x^{0.5})$ 左右，无法正确反映格式在光滑区的理论精度。使用 $\sin(\pi x)$ 可以消除这个干扰，使实测阶数与 Taylor 展开的理论阶数直接对比。

**关键设计：**
- CFL 固定为 0.8，计算至 $t=1.0$
- 实际终点时间 `t_actual = nsteps * dt`（因 dt 与 Δx 绑定，nsteps 取整后终点略有偏移）
- 精确解在 `t_actual` 时刻计算，保证误差比较的公平性
- 使用动态内存分配 (`allocate`/`deallocate`) 适应不同 N

**输出文件：**
| 文件名 | 内容 |
|--------|------|
| `convergence.dat` | 7 列数据：N, dx, L1_LW, Linf_LW, L1_VL, Linf_VL, L1_SB, Linf_SB |

#### `stability_study.f90` — Task 4 数值稳定性验证

**作用：** 用光滑初值，在 5 个不同 CFL 数（0.5, 0.8, 1.0, 1.05, 1.1）下运行三种格式，跟踪 $L_1$ 范数（网格平均绝对解）随时间的变化，检测数值失稳。

**为什么用光滑初值而非方波？**
方波初值下 LW 格式即使在稳定 CFL 也会产生振荡（Gibbs 现象），自动化的发散检测会被混淆。光滑初值下任何大于 O(1) 的增长都是真实的数值失稳信号。

**为什么使用 $L_1$ 范数？**
对于光滑初值 $\sin(\pi x)$，精确解的 $L_1 = \frac{1}{N}\sum|u_i| = 2/\pi \approx 0.6366$ 严格守恒。相比 $\max|u|$，$L_1$ 提供明确的常数值参考线——任何偏离此常数的趋势都是不稳定性的早期信号，网格平均还可平滑单点舍入误差波动。

**失稳判据：**
- $L_1 > 10^{10}$ → 指数增长发散
- $L_1 \neq L_1$（NaN 自比较失败）→ 数值溢出

**输出文件：**
| 文件名 | 内容 |
|--------|------|
| `stability_l1.dat` | 4 列数据：CFL 值, 格式名(LW/VL/SB), 时间 t, $L_1$ |

#### `adaptive_lw.f90` — Task 5 自适应 LW 改进方案

**作用：** 实现带激波传感器的自适应 Lax-Wendroff 格式，与原有三种格式对比。

**核心思想：**

定义一个光滑度指标 $\theta_j$：

$$\theta_j = \frac{|u_{j+1} - 2u_j + u_{j-1}|}{|u_{j+1} - u_j| + |u_j - u_{j-1}| + \varepsilon}$$

- 在间断附近，二阶差商（分子）远大于一阶差商（分母），$\theta_j \to 1$
- 在光滑区域，二阶差商接近零，$\theta_j \to 0$

然后将斜率调整为：
$$\delta_j^{\text{ALW}} = (1 - \theta_j) \cdot (u_{j+1} - u_j)$$

- $\theta_j \approx 1$ → $\delta_j \approx 0$（退化为迎风格式，单调无振荡）
- $\theta_j \approx 0$ → $\delta_j \approx u_{j+1}-u_j$（标准 LW，二阶精度）

**输出文件：**
| 文件名 | 内容 |
|--------|------|
| `adaptive_lw_t2.dat` | ALW 格式在 t=2.0 的解 |
| `adaptive_lw_t8.dat` | ALW 格式在 t=8.0 的解 |
| `adaptive_tv.dat` | ALW 格式的 TV 时间序列 |

### 2. 共享子程序库（1 个）

#### `schemes.inc` — Fortran 公共子程序

**作用：** 被所有 Fortran 主程序通过 `include 'schemes.inc'` 引入。包含 8 个子程序，避免代码重复。

**为何用 `include` 而非 `module`？**
本机 gfortran 15.2.0（MSYS2）在处理模块数组参数时存在内部编译器错误（ICE / Segmentation fault），最小模块可编译但含 `real(8), intent(in) :: u(N)` 等数组哑元的模块会触发崩溃。改用 `include` 方式将子程序直接嵌入主程序的 `contains` 块，功能完全相同且绕过了编译器 bug。

| 子程序 | 功能 |
|--------|------|
| `advance` | 统一 MUSCL 时间推进：$u^{n+1}_j = u^n_j - \nu(u_j-u_{j-1}) - \frac{\nu(1-\nu)}{2}(\delta_j-\delta_{j-1})$ |
| `compute_delta_lw` | Lax-Wendroff 斜率：$\delta_j = u_{j+1} - u_j$（无限制器） |
| `compute_delta_vl` | van Leer 斜率：$\delta_j = \phi_{\text{VL}}(r_j) \cdot (u_{j+1} - u_j)$ |
| `compute_delta_sb` | SUPERBEE 斜率：$\delta_j = \phi_{\text{SB}}(r_j) \cdot (u_{j+1} - u_j)$ |

| `total_variation` | 计算 TV = $\sum_i \vert u_{i+1} - u_i \vert$（含周期卷绕） |
| `exact_solution` | 光滑初值 $\sin(\pi x)$ 对应的精确解 $u(x,t) = \sin(\pi(x-at))$ |
| `compute_errors` | 计算 $L_1 = \frac{1}{N}\sum|u_i-u_i^{\text{exact}}|$ 和 $L_\infty = \max|u_i-u_i^{\text{exact}}|$ |

## 数值方法说明

### 统一框架公式

所有格式使用同一个时间推进公式，仅斜率 $\delta_j$ 的定义不同：

$$u_j^{n+1} = u_j^n - \nu(u_j^n - u_{j-1}^n) - \frac{\nu(1-\nu)}{2}(\delta_j - \delta_{j-1})$$

其中 $\nu = a\Delta t / \Delta x = 0.8$。

### 三种斜率定义

| 格式 | $\delta_j$ | 限制器 $\phi(r)$ | 特点 |
|------|-----------|-----------------|------|
| Lax-Wendroff | $u_{j+1} - u_j$ | 无 | 二阶，有振荡 |
| van Leer | $\phi_{\text{VL}}(r_j)(u_{j+1}-u_j)$ | $(r+\vert r\vert)/(1+\vert r\vert)$ | TVD，均衡 |
| SUPERBEE | $\phi_{\text{SB}}(r_j)(u_{j+1}-u_j)$ | $\max(0,\min(2r,1),\min(r,2))$ | TVD，最陡 |

其中 $r_j = (u_j - u_{j-1})/(u_{j+1} - u_j)$。

### 周期边界处理

域为 $[-1, 1]$，周期长度 $L=2$。边界处通过卷绕索引实现：
- $j=1$ 时：$j-1 \to N$，$j+1 \to 2$
- $j=N$ 时：$j-1 \to N-1$，$j+1 \to 1$

### 解析解

对于方波初值，解析解为 $u(x,t) = u_0(x - at)$ 做周期延拓。由于 $a=1$，在 $t=2.0$ 和 $t=8.0$ 时刻波刚好走完整数个周期，解析解与初值完全重合。

### Task 3：精度阶数

| 格式 | $L_1$ 收敛阶 | $L_\infty$ 收敛阶 |

| Lax-Wendroff | 1.99 | 1.98 |
| van Leer | 2.02 | 1.39 |
| SUPERBEE | 1.88 | 1.10 |

- LW 在两个范数下均达到 $\mathcal{O}(\Delta x^2)$
- VL 和 SB 的 $L_\infty$ 降阶 → 限制器在极值点局部退回一阶（TVD 格式的必要代价）

### Task 4：稳定性条件

| CFL | LW | van Leer | SUPERBEE |
|:---:|:---:|:---:|:---:|
